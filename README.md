## Paper-Download V1.0

```text
简介：
    知网、万方、维普、爱学术下载器
起源：
	1.最近写毛概论文，学校给的知网账号下载有问题。于是就写个自己用。
	2.感谢anypaper作者
安装：
	pip install requests
	windows用户可直接食用，已打包好。
功能：
	如简介所言。后期会加个某主题论文下的爬虫，供大批量下载。不过最近这段时间会忙期末考。
用法:
        图形化界面，TK写的，丑的一批。
```
